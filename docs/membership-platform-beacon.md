# Beacon CRM (~£936/yr excl. VAT)

UK-built charity CRM with membership management. Designed for UK charities and nonprofits.

## Pricing

Starter plan £59.50/mo + Memberships add-on £11/mo + Events & ticketing £7.50/mo = **£78/mo (£936/yr)** on annual billing (10% discount). Excludes VAT. Up to 1,000 contacts. 25 custom fields included.

## Pro/Con

**Pro:**

- UK-native, GBP billing
- REST API
- Stripe integration
- Zero transaction surcharges on payments
- Built for UK charities

**Con:**

- Leans more charity/fundraising than association management
- Memberships and Events are paid add-ons, not included in base price
- Less feature depth than WildApricot or CiviCRM for tiered membership levels and member directories

## OJS Integration

Possible via REST API. Would need investigation.

## Bottom Line

Beacon is a solid UK-native option with zero transaction surcharges and a clean REST API. However, it's less proven for association management than WildApricot or CiviCRM, with membership and events as paid add-ons rather than core features. The weakest of the shortlisted options, but worth keeping in the comparison for its UK roots and straightforward pricing.
